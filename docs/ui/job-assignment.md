## Component Labels

| # | Element | Description |
|---|---------|-------------|
| 1 | Tab Bar | Switch between Annotation and QA assignment |
| 2 | Unassigned Jobs Panel | Left panel with job list (UPLOADED or ANNOTATED) |
| 3 | Search & Dataset Filter | Filter jobs by name and dataset |
| 4 | Jobs Table | Selectable list of unassigned jobs with checkboxes |
| 5 | Select All / Count | Checkbox to select all + selected count display |
| 6 | Assignment Controls Panel | Right panel with strategy and user selection |
| 7 | Strategy Toggle | Radio buttons: Manual / Round-Robin |
| 8 | User Selector (Manual) | Dropdown of active annotators/QA users |
| 9 | "Assign N Selected Jobs" | Primary action button (manual mode) |
| 10 | User Checklist (Round-Robin) | Multi-select with current workload counts |
| 11 | "Preview Distribution" | Opens preview dialog before confirming |
| 12 | Preview Dialog | Shows proposed distribution before confirmation |

## Interactive Elements

| Element | Action | Result |
|---------|--------|--------|
| Annotation/QA Tab | Click | Switches job pool (UPLOADED vs ANNOTATED) and user list (ANNOTATOR vs QA) |
| Job Checkbox | Check/uncheck | Adds/removes job from selection, updates count |
| Select All | Check | Selects all visible jobs |
| Search | Type | Filters jobs by file name |
| Dataset Filter | Select | Filters jobs by dataset |
| Strategy: Manual | Select | Shows single-user dropdown + assign button |
| Strategy: Round-Robin | Select | Shows multi-user checklist + preview button |
| User Dropdown (Manual) | Select | Picks assignee |
| "Assign N Jobs" | Click | Assigns selected jobs to chosen user |
| User Checkboxes (Round-Robin) | Check/uncheck | Includes/excludes user from distribution |
| "Preview Distribution" | Click | Opens preview dialog |
| "Confirm Assignment" | Click (in preview) | Executes round-robin assignment |
